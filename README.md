# Windows Server Home Lab – Active Directory

 

## Overview

Built a virtualized Active Directory environment using VirtualBox to simulate a small business network and develop hands-on IT infrastructure skills.

 

## Environment

- VirtualBox 

- Windows Server 2022 (Domain Controller) 

- Windows 10/11 (Client) 

- Internal Network + NAT 

 

## Architecture

- **DC01 (192.168.10.10):** AD DS, DNS, DHCP 

- **Client VM:** Domain-joined, RSAT tools 

 

## What I Implemented

- Active Directory setup (users, groups, OUs) 

- DNS and DHCP configuration 

- File sharing and permissions 

- Group Policy (GPO) deployment 

- Domain join and client administration 

 

## Troubleshooting Highlights

- Fixed DNS resolution failures (service + client config) 

- Resolved DHCP authorization and scope issues 

- Corrected GPO application via OU and filtering
