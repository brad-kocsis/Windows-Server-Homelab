# Projects
This repository is reserved for the projects started by Brad Kocsis as a way to practice soft skills beneficial in the IT career field.
This project documents the design and deployment of a Windows Server-based Active Directory lab environment built in a virtualized setting.
The environment includes:
Windows Server Domain Controller
Active Directory Domain Services (AD DS)
DNS configuration and troubleshooting
DHCP scope creation and authorization
Organizatoin Unit (OU) design
Security Gruop implementation
Group Policy Object (GPO) deployment and enforcment
Client workstation dopmain join
RSAT administraion from client machine
The lab simulates a small business environment (company.local) with centralized authentication, policy enforcment, and IP adress managment.
Throughout the build, multiple networking and DNS-related issues were encountered and resolved, reinforcing core troubleshooting skills and understanding of Windows infrastructure behavior.
This project demonstrates hands-on experience with:
Domain creation and management
Policy-based administration
Network service configuration
Client-server authentication flow
Infrastruture troubleshooting methodology
